# Vault Diver — website

Static promo site for Vault Diver, served via GitHub Pages at
https://arturduartecruz.github.io/vault-diver-site/

Single self-contained `index.html` (all assets inlined). Publish with
`pwsh -File publish_site.ps1`.

**Game repo:** the Vault Diver game itself (Godot source) lives separately at
`D:\repos\video_game`. Kept as its own repo: different toolchain and deploy
target (Godot → desktop/APK) from this static site.
